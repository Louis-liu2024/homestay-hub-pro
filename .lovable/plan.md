
# 酒店SaaS管理平台 MVP

## 整体架构
- 左侧导航栏（Sidebar）+ 右侧内容区布局
- 多角色权限：管理员、运营人员、普通用户
- Supabase 认证 + 数据库，模拟数据填充
- 中文界面

## 模块一：用户认证与权限
- 登录/注册页面（邮箱+密码）
- 用户角色表（admin / operator / user）
- 路由守卫：未登录跳转登录页

## 模块二：数据池（/data-pool）
- **酒店列表页**：横向可滚动表格，左侧酒店名称列冻结不动，右侧展示评分、渠道来源、房间量、近7天空房率、标签等指标
- 搜索栏：按酒店名称/渠道/标签筛选
- 每行右侧固定操作列（详情、发布按钮），不随横向滚动
- 支持多选（checkbox），底部浮动"一键发布"按钮
- **酒店详情页**（/data-pool/:id）：展示酒店完整信息 + 所有房型列表
- 每个房型展示价格、面积、床型、早餐等信息
- 房型级别的"发布"和"订阅价格"按钮

## 模块三：价格计算器（/price-calculator）
- 配置规则表格：按标签、品牌、时间段设置价格涨幅比例
- 支持新增/编辑/删除规则
- 规则字段：标签、品牌、开始日期、结束日期、涨幅百分比

## 模块四：订单管理（/orders）
- 订单列表：展示用户订房订单（订单号、酒店、房型、入住日期、状态等）
- 订单状态：待领取 → 已领取 → 已完成
- "领取任务"按钮，领取后可填写：
  - 在其他OTA平台下的订单号
  - 联系方式
  - 备注信息

## 模拟数据
- 预填充 20+ 酒店/民宿数据，含多种渠道（携程、美团、Booking等）
- 每个酒店 3-8 个房型
- 10+ 条模拟订单

## 页面路由
- `/login` — 登录
- `/data-pool` — 数据池列表
- `/data-pool/:id` — 酒店详情
- `/price-calculator` — 价格计算器
- `/orders` — 订单管理
